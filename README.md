# 🎂 Felicitación de cumpleaños sorpresa 🐾

Una página web sorpresa, girly y chic, en tonos amarillos, con temática del gato **Yogui**.

## ✨ Qué incluye
- **Pantalla sorpresa** que no parece una felicitación (Yogui "guarda un secreto").
- Explosión de **confeti**, **globos** y emojis flotantes al abrir.
- **Galería de fotos** de Yogui.
- **Pastel interactivo**: soplar las velitas y pedir un deseo.
- **Música de cumpleaños** generada en el navegador (sin archivos externos).
- Mensaje personalizado con mucho cariño.
- 100% responsive (pensada para abrirse en el móvil).

## 🖊️ Personalizar
Edita el bloque `CONFIG` al final de `index.html`:
```js
const CONFIG = {
  nombre: "Amiga",   // nombre de tu amiga
  edad: 25,
};
```
